## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:4321` to see your portfolio.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Customization

All content is managed through `src/config.ts`. Simply edit this file to update:

- Personal information
- Skills
- Projects
- Experience
- Contact details
